To install, download ZIP from github.

To launch, open the "Start 64-bit.bat" or "Start 32-bit.bat" file.

Portable binaries of AutoHotkey are already included in.
Yes, I know that's not how repositories.

To use, press the right alt button while a text field is selected.
If a text field is not selected, the program will freeze. In this case, right click on the tray icon and press reload script.

To access settings, right click on the tray icon.

To make your own preset, make a new text file and save it as .ini in the \presets\ folder.
Learn from the included presets.
